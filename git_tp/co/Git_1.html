Configuration de git

Configuration : Votre identité

Avant votre première utilisation, il est conseillé de configurer Git en indiquant l'identité que vous utiliserez.

Pour cela vous devez définir votre nom et votre email avec les commandes

1
git config --global user.name "VOTRE_NOM"
2
git config --global user.email "VOTRE_EMAIL"
git config --global user.name "VOTRE_NOM"
git config --global user.email "VOTRE_EMAIL"

Installation de Nodepad++

Nous désirons changer l’éditeur par défaut de git (notepad sous windows)

Installer notepad++ : https://notepad-plus-plus.org/fr/

Ajouter notepad++ dans le path

Vérifiez en tapant la commande suivante dans un terminal

1
notepad++
notepad++

L’éditeur notepad++ s'ouvre

Configurer Notepad++ avec git

Créons un nouveau fichier comme suit dans le répertoire racine de l'utilisateur ex : C:\Users\mbeng sur mon ordinateur

1
notepad++ .bash_profile
notepad++ .bash_profile

Ajouter la commande suivante dans le fichier

1
alias npp='notepad++.exe -multiInst -nosession'
alias npp='notepad++.exe -multiInst -nosession'

Enregistrer et quitter

Relancer le terminal Git Bash et tapez la commande suivante :

1
npp
npp

Si la configuration est correcte, léditeur notepad++ s'ouvre.

Fermer l'éditeur.

Nous allons maintenant changer l'éditeur par défaut. Tapez la commande suivante dans le terminal Git Bash

1
cat ~/.gitconfig
cat ~/.gitconfig

Nous obtenons

1
[user]
2
        name = "VOTRE_NOM"
3
        email = "VOTRE_EMAIL"
[user]
        name = "VOTRE_NOM"
        email = "VOTRE_EMAIL"

Dans le terminal Git Bash, tapez la commande

1
git config --global core.editor "notepad++.exe -multiInst -nosession"
git config --global core.editor "notepad++.exe -multiInst -nosession"

Pour vérifier,

1
git config --global --list
git config --global --list

Nous obtenons (avec ma configuration)

1
user.name=bigdata67
2
user.email=bigdata167@gmail.com
3
core.editor=notepad++.exe -multiInst -nosession
user.name=bigdata67
user.email=bigdata167@gmail.com
core.editor=notepad++.exe -multiInst -nosession

Pour vérifier, entrez la commande suivante

1
p4merge
p4merge

P4Merge avec git

1
git config --global merge.tool p4merge
2
git config --global mergetool.p4merge.path "C:/Program Files/Perforce/p4merge.exe"
3
git config --global diff.tool p4merge
4
git config --global difftool.p4merge.path "C:/Program Files/Perforce/p4merge.exe"
5
git config --global difftool.prompt false
6
git config --global mergetool.prompt false
7
