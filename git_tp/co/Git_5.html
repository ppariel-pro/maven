Visualiser les modifications entre le répertoire de travail et la zone de staging

1
npp README.md # ajouter du texte
2
git status
3
git add README.md
4
npp README.md # ajouter du texte
5
git status
6
git diff # montre les modif entre le working dir et stagged dir
7
git difftool # c mieux
npp README.md # ajouter du texte
git status
git add README.md
npp README.md # ajouter du texte
git status
git diff # montre les modif entre le working dir et stagged dir
git difftool # c mieux

Visualiser les modifications entre le répertoire de travail et Git Repository

1
git diff HEAD
2
git difftool HEAD
git diff HEAD
git difftool HEAD

Visualiser les modifications entre la zone de staging et Git Repository

1
git diff --staged HEAD
2
git difftool --staged HEAD
git diff --staged HEAD
git difftool --staged HEAD

Limiter la comparaison à un fichier (ou chemin)

git diff -- README.md

git difftool -- README.md
